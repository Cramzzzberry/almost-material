Colors in Almost Material

Almost Material has a default palette of 7 colors
3 Main Colors:
  • Primary
  • Secondary
  • Tertiary
4 Status Colors:
  • Info
  • Warning
  • Success
  • Error

Custom palette for Almost Material Plugin

You can override the default palette by making your own palette Object and pass it to almostMaterial key under theme like this:
{{ overridePaletteSnippet }}

Extend your palette to Tailwind Classes with different shades

Almost Material Plugin has 10 shades for each color in the palette:
  • Primary-0
  • Primary-10
  • Primary-20
  • Primary-30
  • Primary-40 (Default Shade)
  • Primary-50
  • Primary-60
  • Primary-70
  • Primary-80
  • Primary-90
  • Primary-95
  • Primary-99
  • Primary-100
If you want to extend the palette to Tailwind Classes, Use the function CreateShades(yourPalette) to produce these shades and extend it via theme like this:
{{ extendPaletteSnippet }}
